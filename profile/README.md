### Qué hacemos

Construimos **agentes IA conversacionales** que se integran con los sistemas de tu empresa (CRM, WhatsApp, bases de datos, ERPs) y resuelven consultas de clientes, cualifican leads o automatizan operaciones — sin intervención humana.

**Stack técnico:** Python · LangGraph · LangChain · FastAPI · Weaviate · PostgreSQL · Next.js

### Para quién

| Sector | Caso de uso |
|--------|-------------|
| **Ecommerce** | Atención al cliente 24/7, recuperación de carritos, recomendaciones |
| **Clínicas** | Gestión de citas, consultas frecuentes, recordatorios |
| **Inmobiliarias** | Cualificación de leads, agendamiento de visitas |
| **Restaurantes** | Reservas por WhatsApp, consultas sobre carta y alérgenos |
| **Despachos** | Triaje de casos, cualificación de clientes, agenda de consultas |
